# 네이버웹툰 웹툰 콘텐츠 플랫폼 Back-End 개발 - 기술 역량

## [필수] 보유하신 개발 기술별 역량을 상세히 기술해 주세요. (1000자 이내)

**Java** - 자유자재로 구사 가능. (주)리만코리아와 (주)지엠솔루션에서 Spring Boot 기반 백엔드 개발 3년 이상 경험. Java 21의 최신 언어 기능을 활용한 개인 프로젝트에서 CQRS 패턴 기반 마이크로서비스 아키텍처 설계·구현. 멀티모듈 Gradle 프로젝트에서 6개 이상의 독립적인 API 서버 개발 및 운영 경험.

**Spring Framework (Spring Boot)** - 자유자재로 구사 가능. Spring Boot 기반 RESTful API 서버 설계·구현. (주)리만코리아에서 스페이스 예약시스템 10개 이상 모듈의 백엔드 서버 구축(공통 모듈, DB 연동 모듈, HTTP 클라이언트 모듈, Slack 알림 모듈, AWS S3 연동 클라이언트 모듈, API 모듈 등). Spring Data JPA, Spring Batch, Spring Security 활용한 엔터프라이즈급 애플리케이션 개발 경험.

**Spring Batch** - 실무 프로젝트에서 배치 모듈 구축 및 성능 최적화 경험. (주)리만코리아에서 3-5만건 배치 처리 시간을 120분에서 30분으로 4배 개선(Spring Batch Partitioning Job, Multi Threading, 커스텀 Partitioner 구현). (주)지엠솔루션에서 MSA 환경의 분산 트랜잭션 누락 문제를 ChainedTransactionManager 적용으로 해결. 4,000여 개 대량 데이터 외부 API 연동 배치잡 테스트 시간을 120분 이상에서 3분 내외로 약 40배 성능 개선(비동기 API 호출, 배치 업데이트 적용).

**MyBatis** - 실무 프로젝트에서 복잡한 조회 쿼리 구현 경험. (주)리만코리아와 (주)지엠솔루션에서 MyBatis를 활용한 데이터 접근 계층 구현.

**Spring Data JPA** - 실무 프로젝트에서 JPA를 활용한 엔티티 설계 및 트랜잭션 관리 경험. Querydsl 활용한 동적 쿼리 구현 경험.

**MySQL** - 실무 프로젝트에서 MySQL 데이터베이스 설계 및 운영 경험. (주)리만코리아와 (주)지엠솔루션에서 MySQL 기반 시스템 개발 및 운영.

**Redis** - 실무 프로젝트에서 캐싱 및 세션 관리로 활용. 개인 프로젝트에서 OAuth State 관리, 멱등성 보장 등 다양한 용도로 활용. Spring Data Redis를 활용한 캐시 전략 수립 및 TTL 기반 자동 만료 메커니즘 구현 경험.

**Kafka** - 개인 프로젝트에서 이벤트 기반 CQRS 동기화 메커니즘 구현 경험. EventPublisher와 EventConsumer를 직접 개발하여 Aurora MySQL의 쓰기 작업을 Kafka 이벤트로 발행하고, MongoDB Atlas에 1초 이내 실시간 동기화하는 파이프라인 구축.

**OpenFeign** - 실무 프로젝트에서 외부 API 클라이언트 개발 경험. (주)리만코리아에서 CJ 대한통운 API 연동 시 원데이 토큰 호출 횟수를 5분 내 4,000회 이상에서 1시간에 1번으로 400배 이상 절약하며 운영 안정성 향상.

**Kubernetes** - (주)리만코리아에서 Kubernetes 환경의 운영 경험. ArgoCD를 활용한 CI/CD 파이프라인 구축 및 운영 경험.

**Git** - 실무 프로젝트에서 Git을 활용한 협업 개발 경험. Jira, Slack을 활용한 프로젝트 관리 및 협업 경험.

**AWS** - (주)리만코리아와 (주)지엠솔루션에서 AWS 환경의 애플리케이션 배포 및 운영 경험. AWS S3 연동 클라이언트 모듈 개발 경험.

**MongoDB** - 개인 프로젝트에서 MongoDB Atlas를 활용한 Query Side(읽기) 최적화 구조 설계. Vector Search를 활용한 RAG 챗봇 개발 경험.
